# Backend README

## Instalación

```bash
npm install
```

## Configuración

El archivo `.env` contiene las variables de entorno. Por defecto:
- PORT=3000
- NODE_ENV=development

## Iniciar el servidor

```bash
npm start
```

Para desarrollo con auto-reload:
```bash
npm run dev
```

## Estructura de datos

Todos los archivos JSON están en la carpeta `data/`:
- estudiantes.json
- materias.json
- horarios.json
- noticias.json
- eventos.json
